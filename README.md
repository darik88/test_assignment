1. These tests check simple arithmetic operations with 2 operands on https://bd-testing-app.herokuapp.com/

2. Legal operands are in range [-100, 100].

3. To run these tests you should have Win or Mac PC with installed Chrome Browser.
No other browsers or platforms are supported.

4. To install all required packages type in: 

    "pip install -r requirements.txt"
    
5. To run all tests type in: 

    "pytest -s -v tests\"
